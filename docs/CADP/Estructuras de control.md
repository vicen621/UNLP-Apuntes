## Decisión

    ...
end

## Repetición

```pascal
for i := 1 to 10 do begin
    ...
end;
```

---

## Iteración (While)

```pascal
while (boolean) do begin
    ...
end;
```

---

## Iteración (Repeat)

```pascal
repeat
    ...
until (boolean)
```

## Selección

```pascal
case variable of
    1: begin
        ...
    end;
    2: ...
    3..5: ...
    else: ...
end;
```
